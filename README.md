## 👩‍💻 About

Personal portfolio website of **Sadu Hasini**, a 2nd-year B.Tech Information Technology student at **GVP College of Engineering for Women (GVPCEW)**, Andhra Pradesh.

Built with a **cyberpunk / neon aesthetic** using pure HTML, CSS, and JavaScript — no frameworks needed.

---

## ✨ Features

- ⚡ Animated neon UI with cyberpunk theme
- 🖱️ Custom cursor with smooth trailing effect
- ✍️ Typing animation for roles
- 📱 Fully responsive (mobile + desktop)
- 🌟 Scroll-reveal animations
- 📬 Contact form (powered by Formspree)
- 🎨 Floating particle effects
- 🔤 Glitch text animation

---

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Structure & layout |
| CSS3 | Styling, animations, effects |
| JavaScript | Interactivity & logic |
| Google Fonts | Orbitron, Rajdhani, Share Tech Mono |
| Formspree | Contact form backend |
